About the files format:

To use this notebook it is needed to have Jupyter Lab installed and IRKernel r-package.

Instructions of how to install both can be found here:
https://richpauloo.github.io/2018-05-16-Installing-the-R-kernel-in-Jupyter-Lab/


Otherwise, the content of the notebook can be downloaded in html format and the code in R format from the other_versions/ folder.


Index:

spatial_regression_part1.ipynb - Lecture presentation
